# KotlinCalcPoints

The Kotlin Program calculates the minimum distance of points.

Sample Coordinates: 

![Coordinates](src/main/resources/Coordinates.png "Coordinates")

Minimum distance of all points = 2.0


## Build

gradle build

gradle compileKotlin

gradle compileTestKotlin

./gradlew tasks

gradle test
